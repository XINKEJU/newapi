# 2025-06-12 工作日志

## 俄罗斯本土化 Bug 修复

### Bug #1: RUB 汇率使用 CNY 系数
- **根因**: 系统只有单一 `USDExchangeRate = 90.0` 用于所有货币转换，90 实际是 USD→RUB 但 CNY 也用它
- **修复**: 添加独立的 `CNYExchangeRate` (默认 7.25) 和 `RUBExchangeRate` (默认 90.0)
- **涉及文件**:
  - `setting/operation_setting/payment_setting_old.go` — 新增两个汇率变量
  - `setting/operation_setting/general_setting.go` — `GetUsdToCurrencyRate()` 不再接收 usdToCny 参数，改为直接按 display type 选取对应汇率
  - `controller/billing.go` — CNY 用 CNYExchangeRate, RUB 用 RUBExchangeRate
  - `logger/logger.go` — LogQuota/FormatQuota 新增 RUB 分支 (₽符号)
  - `controller/misc.go` — API 新增 cny_exchange_rate, rub_exchange_rate
  - `model/option.go` — InitOptionMap + updateOptionMap 新增 CNYExchangeRate, RUBExchangeRate

### Bug #2: 前后端默认货币不一致
- **根因**: 后端默认 RUB, 前端 DEFAULT_CURRENCY_CONFIG 默认 USD
- **修复**: 前端 DEFAULT_CURRENCY_CONFIG.quotaDisplayType 从 'USD' 改为 'RUB'
- **前端文件**:
  - `stores/system-config-store.ts` — DEFAULT_CURRENCY_CONFIG 默认 RUB + 新增 cnyExchangeRate/rubExchangeRate
  - `hooks/use-system-config.ts` — API 映射新增 cny/rub 字段
  - `lib/currency.ts` — getDisplayMeta() CNY 用 cnyExchangeRate, RUB 用 rubExchangeRate
  - `features/auth/types.ts` — SystemStatus 新增 cny/rub exchange rate 字段
  - `features/wallet/index.tsx` — effectiveUsdExchangeRate 按 display type 选汇率
  - `features/pricing/hooks/use-pricing-data.ts` — usdExchangeRate 按 display type 选汇率
  - `features/pricing/components/dynamic-pricing-breakdown.tsx` — 添加 RUB 分支
