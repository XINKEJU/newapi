## 克隆并分析 new-api 项目

- 仓库来源：https://github.com/QuantumNous/new-api.git（Calcium-Ion/new-api 的 QuantumNous fork）
- 克隆位置：C:\Users\CORGI\IdeaProjects\newapi

### 项目概要
- **定位**：下一代 LLM API 网关 + AI 资产管理系统（One-API 增强版）
- **版本**：由 Dockerfile 可见 Go 1.26.1，模块声明 go 1.25.1
- **规模**：584 Go 文件 + 907 TSX/TS 文件

### 技术栈
- **后端**：Go + Gin + GORM（支持 MySQL / PostgreSQL / SQLite）
- **缓存**：Redis（频道/Token/用户三级缓存）
- **前端**：React 19 + TanStack Router + TailwindCSS 4 + shadcn/ui，双主题（default/classic），Rsbuild 构建，i18n 5种语言
- **部署**：Docker 多阶段构建（bun 前端 → Go 后端 → debian:slim 运行时），默认端口 3000

### 核心模块
| 目录 | 职责 |
|------|------|
| relay/channel/ | 38+ AI 提供商适配器（OpenAI/Claude/Gemini/DeepSeek/Baidu/Ali...） |
| model/ | GORM 数据模型 + AutoMigrate（24 张表） |
| controller/ | HTTP 控制器层 |
| middleware/ | Auth / RateLimit / i18n / CORS / Gzip |
| service/ | 业务逻辑（Billing/Channel/File/Epay...） |
| router/ | 路由注册（API/Relay/Video/Web） |
| web/ | 前端（default + classic 双主题） |

### 主要功能
- 多渠道管理、自动测试、故障转移、优先级/权重路由
- Token 配额管理、用户分组、兑换码
- 订阅计划（Stripe/Epay/Waffo 支付）
- 异步任务：Midjourney/Suno/Kling/Jimeng/Vidu/Sora 等图/音频/视频生成
- OAuth2（GitHub/Google/OIDC）+ WebAuthn Passkey + 2FA
- Prometheus + Pyroscope 性能监控
- 独立日志数据库（LOG_SQL_DSN）
